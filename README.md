# LivingTheDreamToolkit Linux Autobuild
This repository automatically builds Living The Dream Toolkit for Linux users for convenience.

# Credits
I'm not the creator of this tool, this person did: https://github.com/MadMax1960/LivingTheDreamToolkit

# What is the purpose of this repository?
It checks for new releases of the project and then automatically builds them into a self-contained binary for Linux or Steam Deck users. The original repository only provides Windows builds. 

# AUR
I will soon make an AUR package for this, so it should be even easier to install for Arch users.
